# Code of Conduct

## Our Standard

We want this project to be a respectful and productive space.

Expected behavior:

- Be respectful and constructive.
- Focus feedback on code and outcomes.
- Assume good intent and ask for clarification when needed.

Unacceptable behavior:

- Harassment, insults, or personal attacks.
- Discrimination of any kind.
- Publishing private information without permission.

## Scope

This applies to project spaces, including issues, pull requests, and discussions.

## Enforcement

Project maintainers may edit, remove, or reject content that violates this policy.
Repeated or severe violations may lead to temporary or permanent bans.

## Reporting

If you see abusive behavior, report it privately through GitHub security/advisory channels or direct maintainer contact.
